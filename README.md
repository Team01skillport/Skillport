<h1>スキルポート</h1>
<h3>スキルの売買ができるECサイトです。
ユーザーの使い勝手、アプリ起動から購入への流れをできるだけ単純であるようにします。
スキル売買もしますがカテゴリーごとにSNSのようなユーザー交流場所を作り、
その中にマーケットサービスを提供します。
CtoC（もしくは他社との提携でBtoCに拡張）の環境を整えてユーザーたちに自由に利用させ、講義動画で需要を発生させ、ユーザー同士がそれに応えます。 
</h3>
