<h1>データベースのインポート方法</h1>
  <ol>
    <li>XAMPPを起動させる</li>
    <li>mysqlにログインする</li>
    <li>CREATE DATABASE skillport_db; <p>をmysqlに入力する</p></li>
    <li>schema.sqlの内容を全コピー(Ctrl+A)して、mysqlに貼り付けて、Enterを押す</li>
    <li>SHOW TABLES; <p>でテーブルがちゃんと作成できたかを確認</p></li>
    <li>テーブルの情報を抽出してみる</li>
    <li>問題なければこれで終わり</li>
  </ol>
